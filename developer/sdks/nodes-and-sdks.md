# SDKs & node roles

## Node roles (from Yellowpaper / node kit)

* **Public** — RPC, read-heavy
* **Private execution** — HELIX paths
* **Strategy** — Automated routing participants
* **Compliance** — Policy-enforced endpoints
* **DePIN** — Inner company infrastructure
