# Institutional data layer

## Public vs institutional data

| Layer | Style | Examples |
| --- | --- | --- |
| Public | Dune-style open queries | Community dashboards |
| Institutional | Artemis-style KPI models | Capital flows, cohort metrics |

## APIs

* Public tier — rate-limited, community
* Private tier — authenticated, SLA-backed
* Institutional — snowflake-style schema exports (roadmap)
